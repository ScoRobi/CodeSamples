Problem Statement
Please provide a code sample by writing a Java program that determines if a string is a palindrome.  You must write the code yourself – do not use code downloaded from the Internet.   Specific instructions are below:

1)	Read in a text file containing the following data:

<?xml version="1.0"?>
<dataset>
<data1>Don't nod</data1>
<data2>Dogma: I am God</data2>
<data8>Was it Eliot's toilet I saw?</data8>
<data3>Never odd or even</data3>
<data10>How are you era who</data10>
<data4>Too bad – I hid a boot</data4>
<data5>Rats live on no evil star</data5>
<data7>No trace; not one carton</data7>
<data9>Murder for a jar of red rum</data9>
<data11>Madam, I&m Adam</data11>
<test>This is not data</test>
</dataset>

2)	Loop through each line of input data and determine if the data contains a palindrome, ignoring punctuation and capitalization.

3)	Write a XML answer file of the following form, where “true” indicates that the corresponding data element is a palindrome, or “false” otherwise:

<?xml version="1.0"?>
<answers>
<answer1>false</answer1>
<answer2>true</answer2>
<answer8>true</answer8>
…
</answers>

4)	Provide the following :
a.	A jar file of the application along with instructions to execute it.
b.	 Source code.
c.	 An output file containing the data generated by the program given the input file specified above.
